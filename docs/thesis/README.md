## Suggested Thesis Title

> **MedFabric: A Web-Based CT Scan Labeling System for Ischemic Stroke Assessment Using an Event-Driven Streamlit Architecture**

---

## Key Contributions to Highlight

1. **Novel Dispatcher Pattern for Streamlit** - Solving state management in stateless UI framework
2. **Medical Imaging Workflow** - ASPECTS scoring integration for stroke assessment
3. **Multi-Set Labeling Session** - Batch processing with validation tracking
4. **Modular Architecture** - Clean separation of concerns across layers
