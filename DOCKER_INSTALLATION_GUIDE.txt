QUICK START (One-Minute Setup)
================================================================================

  bash docker/install-docker.sh && newgrp docker
  bash docker/setup-env.sh
  bash docker/build.sh
  bash docker/run-dev.sh

Then open: http://localhost:8501


DETAILED STEPS
================================================================================

1. INSTALL DOCKER
   ─────────────────

   bash docker/install-docker.sh

   This script:
   • Detects your OS (Ubuntu, Debian, Fedora, RHEL, CentOS)
   • Installs Docker and Docker Compose
   • Configures Docker daemon
   • Adds you to docker group

   After installation:
   └─ Log out and log back in (or: newgrp docker)

   Verify:
   └─ docker --version
   └─ docker-compose --version


2. SETUP ENVIRONMENT
    ──────────────────

   bash docker/setup-env.sh

   This script:
   • Creates .env configuration file
   • Configures database (SQLite by default)
   • Optional: Tailscale Auth Key for production
   • Creates required directories
   • Validates Docker setup

   Defaults:
   • DATABASE_URL: sqlite:///./medfabric.db
   • TS_AUTH_KEY: (optional, for production)

   Just press ENTER for development setup.


3. BUILD DOCKER IMAGE
    ─────────────────

   bash docker/build.sh

   This script:
   • Builds MedFabric Docker image
   • Installs all dependencies
   • Validates build success

   Takes: ~5-10 minutes on first build

   Options:
   • bash docker/build.sh --no-cache     (rebuild everything)
   • bash docker/build.sh --force        (remove old images)


4. RUN THE APPLICATION
    ──────────────────

   bash docker/run-dev.sh

   This script:
   • Starts the Streamlit application
   • Enables code auto-reloading
   • Shows logs on terminal
   • Listens on http://localhost:8501

   Press Ctrl+C to stop

DEVELOPMENT WORKFLOW
================================================================================

Terminal 1 - Start Application:
  $ bash docker/run-dev.sh
  
  Now running at http://localhost:8501
  Press Ctrl+C to stop

Terminal 2 - Follow Logs:
  $ bash docker/logs.sh dev -f
  
  Shows real-time application logs

Terminal 3 - Edit Code:
  $ vim medfabric/main.py
  
  Changes automatically reload in browser!


PRODUCTION SETUP (WITH TAILSCALE)
================================================================================

1. Get Tailscale Auth Key:
   https://login.tailscale.com/admin/authkeys

2. Edit .env file:
   TS_AUTH_KEY=tskey-auth-XXXXX...

3. Build and run production:
   bash docker/build.sh
   bash docker/run-prod.sh

4. Monitor:
   bash docker/logs.sh prod -f
   bash docker/status.sh


TROUBLESHOOTING
================================================================================

Docker daemon not running?
  sudo systemctl start docker
  sudo systemctl enable docker

Permission denied errors?
  sudo usermod -aG docker $USER
  newgrp docker

Port 8501 already in use?
  bash docker/stop.sh all

Build failed?
  bash docker/build.sh --no-cache

Configuration issues?
  bash docker/status.sh


USEFUL COMMANDS
================================================================================

Check Docker status:
  bash docker/status.sh

View application logs:
  bash docker/logs.sh dev -f

Stop containers:
  bash docker/stop.sh dev

Clean up old images:
  bash docker/clean.sh --all --confirm

Execute command in container:
  docker exec medfabric_app_dev python -c "print('Hello')"

Interactive shell in container:
  docker exec -it medfabric_app_dev /bin/bash
